# Подкаст-плеер

Приложение для поиска и прослушивания подкастов.

## Библиотеки/инструменты

* React
* Redux (thunk, reselect)
* React Router
* PropTypes

## Функционал

* Поиск подкастов
* Подписка на подкасты
* Воспроизведение эпизодов
* Сохранение истории воспроизведения и подписок в localStorage
